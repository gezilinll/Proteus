## 📚 技术栈总览

本项目涉及以下核心技术：

| 分类 | 技术 | 说明 | 文档 |
|------|------|------|------|
| **语言** | TypeScript | 带类型系统的 JavaScript | [→ 学习](./typescript.md) |
| **UI 框架** | React 19 | 声明式 UI 组件库 | [→ 学习](./react.md) |
| **状态管理** | Zustand | 轻量级状态管理 | [→ 学习](./zustand.md) |
| **构建工具** | Vite | 下一代前端构建工具 | [→ 学习](./vite.md) |

## 🎯 学习路径建议

### 入门路径（适合前端新人）

```
TypeScript 基础 → React 核心 → Zustand 状态管理 → Vite 构建
```

### 进阶路径（适合有经验的开发者）

```
React 19 新特性 → Yjs 协同原理 → Canvas 渲染优化 → Monorepo 架构
```

### 工程化路径（适合关注代码质量）

```
ESLint 9 配置 → Vitest 测试 → pnpm 工作区 → CI/CD 集成
```
