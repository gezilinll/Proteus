# Proteus

**AI-Native Multimodal Collaborative Editor**

> Proteus（普罗透斯）：希腊神话中的海神，能随意变换成任何形态。正如我们的编辑器，能承载任何类型的内容。

---

## 我们要解决什么问题

过去十年，创作工具经历了三次范式转移：

| 时代 | 代表产品 | 核心突破 |
|------|----------|----------|
| 2010s 早期 | Sketch | 专为界面设计的工具 |
| 2010s 中期 | Figma | 实时协作 + 浏览器原生 |
| 2020s | Notion, Miro | 多模态内容 + 无限画布 |

2024 年，我们正在进入第四个时代：**AI 原生**。

Cursor 重新定义了代码编辑器。Midjourney 重新定义了图像创作。但在多模态内容编辑领域，AI 仍然是「附加功能」——一个侧边栏、一个按钮、一次调用。

**Proteus 的目标是：从第一行代码开始，就把 AI 作为核心设计约束。**

这不是在现有编辑器上「加 AI」，而是重新思考：当 AI 成为用户的协作者时，编辑器应该长什么样？

---

## 产品定位

Proteus 是一个开源的多模态协同编辑器，面向需要处理复杂内容组合的创作者和团队。

| 维度 | 定位 |
|------|------|
| 内容形态 | 文字 + 图片 + 视频 + 网页 + 矢量路径 + ...，统一画布 |
| 交互方式 | 传统操作 + 自然语言 + 语音指令 |
| 协作模式 | 实时编辑 + 视频对话 + AI 助手 |
| 开放性 | 完全开源，可嵌入、可扩展、可自托管 |

### 目标用户

- **内容创作者**：需要在一个空间里组织文字、图片、视频、参考资料的人
- **产品/设计团队**：需要快速拼接原型、整理需求、进行协作评审的团队
- **教育/培训场景**：需要多人实时标注、讨论、演示的场景
- **开发者**：需要一个可嵌入、可扩展的编辑器内核

## 核心功能

### 1. 多模态画布

一个统一的无限画布，容纳所有类型的内容：

| 元素类型 | 能力 |
|----------|------|
| 文字 | 富文本、Markdown、代码块、公式 |
| 图片 | 导入、裁剪、滤镜、AI 描述、AI 编辑 |
| 视频 | 预览、帧级 Seek、时间轴标注、AI 摘要 |
| 音频 | 波形可视化、转录、AI 摘要 |
| 网页 | iframe 嵌入、实时预览、截图归档 |
| 矢量 | 贝塞尔路径、自由绘制、形状工具 |
| 组件 | 可复用模块、嵌套结构、跨画布引用 |

所有元素共享统一的变换、对齐、分组能力。无需在不同工具之间切换。

### 2. AI Agent 深度集成

不是一个侧边栏，而是一个真正的协作者。

**自然语言操作**

```
"把右边那三张图片缩小一半，排成一列"
"给这个视频的第 30 秒处加个标注"
"把所有红色的元素改成蓝色"
```

Agent 理解空间位置、上下文指代、批量操作。

**多模态理解**

```
"这张图里有什么？"  → 图片内容分析
"帮我总结这个视频的要点" → 关键帧提取 + 内容摘要
"根据这段文字生成配图" → 文生图 + 自动插入
```

Agent 能看懂画布上的内容，并基于理解进行操作。

**对话式迭代**

```
用户：帮我设计一个简洁的名片
Agent：[生成初版]
用户：字再大一点，颜色换成深蓝色
Agent：[应用修改]
用户：不对，我说的是标题的字
Agent：[精确修改标题]
```

保持上下文，支持连续对话式的设计迭代。

### 3. 实时协同

超越「多人编辑」的协同体验：

| 能力 | 说明 |
|------|------|
| 实时同步 | 毫秒级延迟，冲突自动解决 |
| 光标共享 | 看到协作者的位置和选区 |
| 评论标注 | 针对任意元素的评论和讨论 |
| 跟随模式 | 跟随某个协作者的视角浏览 |
| 视频对话 | 内置音视频通话，边聊边编辑 |
| 语音指令 | 说话即操作，AI 实时响应 |
| 离线编辑 | 断网可用，重连自动合并 |

### 4. 开放架构

为嵌入和扩展而设计：

| 能力 | 说明 |
|------|------|
| 独立内核 | 编辑器核心与 UI 解耦，可嵌入任意应用 |
| 插件系统 | 自定义元素类型、工具、面板 |
| API 完备 | 所有操作可通过 API 调用 |
| 自托管 | 完整的后端服务可自行部署 |
| 数据导出 | JSON / 图片 / 视频 / PDF 多格式导出 |

## 与竞品的差异

| 产品 | 强项 | Proteus 的差异 |
|------|------|----------------|
| Figma | 专业设计、组件系统 | 多模态内容支持、AI 原生 |
| Miro | 白板协作、便签 | 更强的媒体处理能力（视频/音频） |
| Canva | 模板丰富、易用 | 可编程、可嵌入、完全开源 |
| Notion | 块编辑、数据库 | 无限画布、空间化布局 |
| Excalidraw | 简洁、开源 | 多模态、AI 集成、视频协同 |

Proteus 不是要替代这些产品。我们的差异化在于：

1. **AI 不是功能，是交互范式**：自然语言和传统操作同等重要
2. **多模态是一等公民**：视频、音频不是附件，是可编辑的元素
3. **协同不止于编辑**：视频对话 + 语音指令 + 实时画布

### Phase 1: 基础编辑器（第 1-4 周）

**目标**：搭建可用的基础编辑器，验证核心架构。

**产出**
- 无限画布 + 视口控制（缩放、平移）
- 基础元素（矩形、圆形、文字、图片）
- 选择、变换、图层、对齐
- Undo/Redo
